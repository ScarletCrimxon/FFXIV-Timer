# ✦ FFXIV Server Status Bot ✦

Live Final Fantasy XIV server status tracking for Discord.

## ✨ Features

- 🟢 Server Online / 🔴 Maintenance Active
- 🛠 Auto-detected All Worlds maintenance (Lodestone RSS)
- 🕒 Daily Reset timer (15:00 UTC)
- 🗓 Weekly Reset timer (Tuesday 08:00 UTC)
- 🌍 Multi-server support

## ⚙ Setup

1. Invite the bot to your server  
2. Run the slash command:
/setup  ->  channel:#your-channel-name

3. The bot will post a live-updating status panel in that channel.

---

## 🔐 Permissions Required

- View Channels  
- Send Messages  
- Embed Links  
- Read Message History  

No administrator permission required.
